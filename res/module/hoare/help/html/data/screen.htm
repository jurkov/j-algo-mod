Der Arbeitsbereich

Der Startbildschrim


      Hiermit öffnet man ein neues c0 Programm, welches direkt mit dem ersten Knoten initialisiert wird.
      Hiermit öffnet man den Arbeitsbildschirm und kann dort ein neues c0 Programm eingeben.
      Hiermit öffnet man eine .jalgo Datei, welche initialisiert wird.
